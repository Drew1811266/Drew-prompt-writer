# Drew-prompt-writer

专为图生图/图生视频场景设计的 AI 提示词生成技能。

## 简介

Drew-prompt-writer 是一个帮助用户快速生成高质量 AI 图像/视频生成提示词的技能。它采用「一级提示词 → 二级提示词」的双层架构，将用户的碎片化描述转换为标准化的结构化提示词。

## 核心功能

- **图生图提示词生成**（暗号：`i`）
- **图生视频提示词生成**（暗号：`v`）
- **多轮对话编辑**（暗号：`c`）
- **结束对话**（暗号：`end` 或 `结束`）

## 暗号系统

| 暗号 | 功能 | 示例 |
|------|------|------|
| `i` | 生成图生图提示词 | `美女 大光圈 85mm i` |
| `v` | 生成图生视频提示词 | `女人 转头 微笑 v` |
| `c` | 修改当前提示词 | `景别改为中景 c` |
| `end` / `结束` | 清空上下文 | `end` |

## 一级提示词格式

一级提示词是用户直接输入的碎片化词块集合：

```
美国 白人 女孩 25岁 上身穿白色短袖 下身穿浅蓝色牛仔短裤 脚穿凉鞋 夏威夷海滩 沙滩上漫步 横屏 全景 50mm焦段 小光圈 户外阳光充足
```

### 支持的元素

- **主体描述**：年龄、国籍、外貌特征
- **服装**：上衣、裤子、裙子、鞋子等
- **场景**：海滩、公园、咖啡厅、室内等
- **动作**：漫步、站立、坐着、跑步等
- **构图**：横屏/竖屏、景别（特写/全景等）
- **摄影术语**：焦段、光圈、镜头运动、角度等
- **图片引用**：`图1中的上衣`、`图2中的裤子` 等

## 图片引用功能

支持引用参考图片中的服装/产品：

| 引用格式 | 示例 | 说明 |
|----------|------|------|
| 图+数字 | `图1中的上衣` | 最常见格式 |
| 图+中文数字 | `图一中的裤子` | 中文数字 |
| 照片+数字 | `照片1中的裙子` | 替代说法 |
| IMG+数字 | `IMG1中的外套` | 英文格式 |

### 引用类型

- **服装类**：上衣、裤子、裙子、套装、鞋子 → `wearing [item] from image X`
- **产品类**：水杯、枕头、小推车 → `holding [item] from image X`
- **配饰类**：包包、首饰、手表 → `carrying [item] from image X`

## 分镜头功能

在图生视频模式下，支持生成分镜头脚本：

**触发方式**：在提示词中加入 `设计X个镜头`

```
女人 转头 微笑 设计三个镜头 v
```

**输出格式**：每个分镜头以一整段连贯文字输出，并入总提示词中。

## 版式规范

本技能采用**模块化 + 灵活组合**的版式设计：

### 标准段落结构

**基础段落**：`(主体)` → `(服装)` → `(配饰)` → `(动作)` → `(场景)` → `(构图)`

**技术段落**：`(镜头)` → `(技术)` → `(光照)` → `(氛围)` → `(保护)` → `(质量)`

**视频专属**：`(分镜头1)` → `(分镜头2)` → `(分镜头3)` → ...

### 版式特点

- **动态增减**：根据用户需求自动调整段落
- **标识符统一**：使用英文括号 `(标识符)`
- **跨设备一致**：确保不同设备上生成的提示词结构相似

## 使用示例

### 示例 1：基础图生图

**输入**：
```
美国 白人 女孩 25岁 上身穿白色短袖 下身穿浅蓝色牛仔短裤 脚穿凉鞋 夏威夷海滩 沙滩上漫步 横屏 全景 50mm焦段 小光圈 户外阳光充足 i
```

**输出**：结构化的英文提示词 + 中文翻译

### 示例 2：带图片引用

**输入**：
```
美国 女孩 25岁 身穿 图1中的上衣 手拿 图2中的奶茶 咖啡厅 柔光 侧拍 i
```

**输出**：包含 `[top from image 1]`、`[bubble tea from image 2]` 等引用标记

### 示例 3：图生视频 + 分镜头

**输入**：
```
女人 转头 微笑 设计三个镜头 v
```

**输出**：包含 (分镜头1)、(分镜头2)、(分镜头3) 的完整提示词

### 示例 4：多轮对话编辑

**第一轮**：
```
22岁 美国 白人 女孩 身穿图1中的连衣裙 公园 散步 v
```

**第二轮整改**：
```
场景改为校园 c
```

**结果**：场景更新为校园，其他元素保持不变

## 安装指南

### 环境要求

- **编辑器**：Trae、VS Code 或任何支持 Skill 的 AI 助手
- **文件编码**：UTF-8
- **系统支持**：macOS、Windows、Linux

### 安装步骤

1. **创建技能目录**
   ```bash
   mkdir -p ~/.trae/skills/Drew-prompt-writer
   ```

2. **复制技能文件**
   ```bash
   cp SKILL.md ~/.trae/skills/Drew-prompt-writer/
   ```

3. **验证安装**
   - 打开 Trae/VS Code
   - 输入测试提示词：`美女 大光圈 i`
   - 检查是否正常生成二级提示词

### 跨设备同步

**方法 1：Git 同步**
```bash
# 克隆仓库到新设备
git clone https://github.com/yourusername/Drew-prompt-writer.git

# 复制到技能目录
cp Drew-prompt-writer/SKILL.md ~/.trae/skills/Drew-prompt-writer/
```

**方法 2：手动复制**
- 将 `SKILL.md` 文件通过云盘/邮件传输到新设备
- 放置到对应目录即可

2. 输入相同的一级提示词
3. 生成的二级提示词将保持相似的结构和专业风格
