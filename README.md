# Drew-prompt-writer

专为图生图/图生视频场景设计的 AI 提示词生成技能。

## 简介

Drew-prompt-writer 是一个帮助用户快速生成高质量 AI 图像/视频生成提示词的技能。它采用「一级提示词 → 二级提示词」的双层架构，将用户的碎片化描述转换为标准化的结构化提示词。

## 核心功能

- **图生图提示词生成**（暗号：`i`）
- **图生视频提示词生成**（暗号：`v`）
- **多轮对话编辑**（暗号：`c`）
- **结束对话**（暗号：`end` 或 `结束`）

## 暗号系统

| 暗号 | 功能 | 示例 |
|------|------|------|
| `i` | 生成图生图提示词 | `美女 大光圈 85mm i` |
| `v` | 生成图生视频提示词 | `女人 转头 微笑 v` |
| `c` | 修改当前提示词 | `景别改为中景 c` |
| `end` / `结束` | 清空上下文 | `end` |

## 一级提示词格式

一级提示词是用户直接输入的碎片化词块集合：

```
美国 白人 女孩 25岁 上身穿白色短袖 下身穿浅蓝色牛仔短裤 脚穿凉鞋 夏威夷海滩 沙滩上漫步 横屏 全景 50mm焦段 小光圈 户外阳光充足
```

### 支持的元素

- **主体描述**：年龄、国籍、外貌特征
- **服装**：上衣、裤子、裙子、鞋子等
- **场景**：海滩、公园、咖啡厅、室内等
- **动作**：漫步、站立、坐着、跑步等
- **构图**：横屏/竖屏、景别（特写/全景等）
- **摄影术语**：焦段、光圈、镜头运动、角度等
- **图片引用**：`图1中的上衣`、`图2中的裤子` 等

## 图片引用功能

支持引用参考图片中的服装/产品：

| 引用格式 | 示例 | 说明 |
|----------|------|------|
| 图+数字 | `图1中的上衣` | 最常见格式 |
| 图+中文数字 | `图一中的裤子` | 中文数字 |
| 照片+数字 | `照片1中的裙子` | 替代说法 |
| IMG+数字 | `IMG1中的外套` | 英文格式 |

### 引用类型

- **服装类**：上衣、裤子、裙子、套装、鞋子 → `wearing [item] from image X`
- **产品类**：水杯、枕头、小推车 → `holding [item] from image X`
- **配饰类**：包包、首饰、手表 → `carrying [item] from image X`

## 分镜头功能

在图生视频模式下，支持生成分镜头脚本：

**触发方式**：在提示词中加入 `设计X个镜头`

```
女人 转头 微笑 设计三个镜头 v
```

**输出格式**：每个分镜头以一整段连贯文字输出，并入总提示词中。

## 使用示例

### 示例 1：基础图生图

**输入**：
```
美国 白人 女孩 25岁 上身穿白色短袖 下身穿浅蓝色牛仔短裤 脚穿凉鞋 夏威夷海滩 沙滩上漫步 横屏 全景 50mm焦段 小光圈 户外阳光充足 i
```

**输出**：结构化的英文提示词 + 中文翻译

### 示例 2：带图片引用

**输入**：
```
美国 女孩 25岁 身穿 图1中的上衣 手拿 图2中的奶茶 咖啡厅 柔光 侧拍 i
```

**输出**：包含 `[top from image 1]`、`[bubble tea from image 2]` 等引用标记

### 示例 3：图生视频 + 分镜头

**输入**：
```
女人 转头 微笑 设计三个镜头 v
```

**输出**：包含 (分镜头1)、(分镜头2)、(分镜头3) 的完整提示词

### 示例 4：多轮对话编辑

**第一轮**：
```
22岁 美国 白人 女孩 身穿图1中的连衣裙 公园 散步 v
```

**第二轮整改**：
```
场景改为校园 c
```

**结果**：场景更新为校园，其他元素保持不变

## 安装使用

1. 将 `SKILL.md` 文件复制到您的 `.trae/skills/Drew-prompt-writer/` 目录
2. 在 Trae 或其他支持 Skill 的 AI 助手中使用
3. 按照「一级提示词 + 暗号」的格式输入需求

## 输出格式

所有输出统一为专业相机/电影机拍摄的写实质感画面：

### 英文提示词结构
```
(主体) [描述]
(场景) [描述]
(构图) [描述]
(镜头) [描述]
(技术) [描述]
(光照) [描述]
(氛围) [描述]
(保护) [保护性提示词]
(质量) [质量标签]
```

### 风格标签
- **图生图**：`professional photography, DSLR quality, realistic textures, natural skin tones`
- **图生视频**：`cinematic film quality, ARRI Alexa style, professional cinematography`

## 技术特点

- **智能扩展**：自动补充样貌、场景、动作细节
- **导演思维**：主动设计光影、色彩、情绪、镜头语言
- **摄影术语映射**：将中文口语转换为标准英文术语
- **保护性提示词**：自动添加肢体保护和参考一致性保护
