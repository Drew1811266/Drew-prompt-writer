# 导演分镜头创作规格

## Why

在图生视频场景中，用户有时只想提供故事/场景的核心描述，希望 AI 像导演一样帮助设计具体的镜头方案。例如用户说"设计三个镜头"或"设计两个镜头（特写、全景）"，AI 需要根据用户的创作意图，自主设计出符合叙事逻辑的镜头组合（远景建立、中景叙事、特写强调等）。

## What Changes

- **新增分镜头识别**: 识别用户输入中的分镜头数量和类型要求
- **新增分镜头创作**: AI 自主设计符合叙事的镜头组合
- **新增镜头序列输出**: 按叙事顺序输出多个镜头方案
- **更新 SKILL.md**: 将分镜头创作功能整合到图生视频部分

## Impact

- Affected specs: 图生视频扩展
- Affected code: `.trae/skills/prompt-crafter/SKILL.md`

## ADDED Requirements

### Requirement: 分镜头识别规则

系统 SHALL 能够识别以下分镜头指令模式：

| 指令模式 | 示例 | 说明 |
|----------|------|------|
| 设计 + 数量 + 镜头 | 设计三个镜头、设计两个镜头 | 仅指定数量 |
| 设计 + 数量 + 镜头（类型列表） | 设计两个镜头（特写、全景） | 指定数量和类型 |
| 生成 + 数量 + 镜头 | 生成三个镜头 | 替代说法 |

### Requirement: 镜头类型定义

| 镜头类型 | 说明 | 英文描述 |
|----------|------|----------|
| 远景 | 建立环境、交代场景 | establishing shot, wide shot, long shot |
| 全景 | 显示全部主体和环境 | full shot, wide view |
| 中景 | 叙事重心、人物全身或大半身 | medium shot, waist-up shot |
| 近景 | 突出主体、面部表情 | close-up shot, bust shot |
| 特写 | 强调细节、情绪 | extreme close-up, detail shot |
| 侧面 | 展示轮廓、运动方向 | side view, profile shot |
| 背面 | 展示姿态、运动 | back view, rear shot |

### Requirement: 分镜头创作逻辑

**WHEN** 用户输入包含分镜头指令
**THEN** AI 按照以下逻辑创作分镜头：

1. **理解核心意图**: 分析用户的场景和主体描述
2. **确定镜头数量**: 按照用户指定的数量设计
3. **选择镜头类型**: 根据场景逻辑选择合适的镜头类型组合
4. **设计镜头序列**: 按叙事逻辑排列镜头顺序
5. **补充每个镜头描述**: 为每个镜头添加具体画面描述

**标准镜头序列逻辑**:
```
镜头1: 远景/建立镜头 - 交代环境、引入场景
镜头2: 中景/全景 - 展示主体和环境关系
镜头3: 近景/特写 - 强调主体细节/表情/动作
（根据数量继续添加）
```

### Requirement: 分镜头输出格式

**WHEN** 分镜头创作完成后
**THEN** 按以下格式输出：

```markdown
### 分镜头序列

**镜头 1: [远景/建立]**
(主体) [描述]
(场景) [描述]
(构图) [描述]
(镜头) [描述]
...

**镜头 2: [中景]**
(主体) [描述]
...
```

### Requirement: 分镜头创作示例

**用户输入**:
```
女人 转头 微笑 设计三个镜头 v
```

**分镜头创作输出**:

```
**镜头 1: 远景建立**
女人站在窗边，自然光照射，夕阳氛围，wide establishing shot，景深涵盖整个房间

**镜头 2: 中景叙事**
女人开始转头，长发随风，side view，smooth tracking movement，柔和的眼神

**镜头 3: 特写强调**
女人微笑的特写，gentle smile，eyes slowly opening，extreme close-up on face，浅景深
```

**用户输入**:
```
海边 女孩 奔跑 设计两个镜头（特写、全景） v

**分镜头创作输出**:

```

## 技术约束

1. 分镜头创作仅在图生视频场景（暗号 `v`）时触发
2. 分镜头数量必须与用户指定一致
3. 如果用户指定了镜头类型，则必须包含
4. 每个镜头描述应包含主体、场景、构图、镜头运动等要素
5. 镜头序列应符合叙事逻辑（远景→中景→特写）
