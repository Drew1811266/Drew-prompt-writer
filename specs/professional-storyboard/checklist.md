# Checklist - 好莱坞专业分镜头脚本规范规格

## 验收检查点

- [x] **标准字段**: 包含镜头号、机位、景别、运动、画面内容、角色动作、构图，光照、时长、备注等字段
- [x] **机位类型定义**: 包含平视、俯视、仰视、正面、侧面、背面、斜角度的定义
- [x] **景别定义**: 包含极远景、远景、全景、中全景、中景、近景、特写的定义
- [x] **相机运动定义**: 包含固定、推、拉、摇、倾、移、跟、环绕、升降、手持的定义
- [x] **角色动作描述规范**: 包含姿态、位置、方向、动作、表情、细节等维度
- [x] **构图原则**: 包含三分法、头部空间、负空间、前景/背景、视线引导
- [x] **专业格式模板**: 包含完整的好莱坞风格分镜头脚本模板
- [x] **专业示例**: 包含至少 1 个完整的好莱坞风格分镜头示例
