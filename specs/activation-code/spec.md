# 暗号触发功能扩展规格

## Why

用户在进行图生图或图生视频时，希望通过简单的暗号快速触发对应的提示词生成功能，无需显式说出"帮我生成提示词"等指令。通过在输入末尾添加特定字母（`i` 代表 image/img2img，`v` 代表 video/img2video），可以快速激活技能并自动识别用户意图。

## What Changes

- **新增暗号触发机制**: 通过结尾字母自动识别触发技能
- **扩展触发条件**: 在现有触发条件基础上新增暗号识别
- **更新 SKILL.md**: 将暗号触发机制整合到技能描述中

## Impact

- Affected specs: prompt-crafter 技能
- Affected code: `.trae/skills/prompt-crafter/SKILL.md`

## ADDED Requirements

### Requirement: 暗号触发定义

暗号是用户在一级提示词末尾添加的特殊字母标记，用于自动触发对应的提示词生成技能。

**暗号规则**:

| 暗号 | 触发技能 | 适用场景 |
|------|----------|----------|
| `i` | 图生图提示词生成 | 用户输入末尾添加空格 + i |
| `v` | 图生视频提示词生成 | 用户输入末尾添加空格 + v |

### Requirement: 暗号识别规则

**触发条件**:
- 用户输入以单个字母 `i` 或 `v` 结尾
- 字母前必须有空格分隔
- 字母单独存在，不与其他字符连接

**识别示例**:

| 用户输入 | 识别结果 | 触发技能 |
|----------|----------|----------|
| `美国 女孩 25岁 沙滩 漫步 i` | 末尾暗号 `i` | 图生图 |
| `女人 转头 微笑 升格镜头 v` | 末尾暗号 `v` | 图生视频 |
| `美女 大光圈 85mm焦段 i` | 末尾暗号 `i` | 图生图 |
| `美国白人女孩i` | 无法识别（无空格分隔） | 不触发 |

### Requirement: 暗号处理流程

**WHEN** 用户输入以暗号结尾
**THEN** 系统执行以下处理：

1. **暗号检测**: 检查输入末尾是否包含空格 + 单个暗号字母
2. **暗号分离**: 从输入中分离出暗号部分
3. **技能选择**: 根据暗号类型选择对应的提示词生成技能
4. **主体提取**: 提取除暗号外的实际提示词内容
5. **技能执行**: 对主体内容执行一级→二级提示词转换
6. **结果输出**: 输出转换后的二级提示词

### Requirement: 暗号自动清除

**WHEN** 暗号被识别并处理
**THEN** 输出的二级提示词中不应包含暗号标记

**示例**:

**用户输入**:
```
美女 大光圈 85mm焦段 特写 仰视角度 i
```

**处理后输出（不含暗号）**:
```
正面提示词:
(主体) beautiful woman, attractive female, elegant pose
(镜头) 85mm lens, portrait lens
(技术) wide aperture (f/1.4-f/2.8), shallow depth of field
(角度) low angle, from below
...
```

## MODIFIED Requirements

### Requirement: 技能触发条件扩展

**原触发条件**:
- 用户以词块形式输入创作需求（一级提示词）
- 请求"帮我生成图片提示词"
- 请求"写一个视频提示词"

**扩展触发条件**:
- 用户输入末尾包含暗号 `i`
- 用户输入末尾包含暗号 `v`

### Requirement: 响应流程扩展

**当收到用户输入且末尾包含暗号时**：

1. **暗号检测**: 检测输入末尾的暗号字母
2. **暗号分离**: 分离出暗号，得到实际提示词主体
3. **技能路由**: 根据暗号类型选择图生图(i)或图生视频(v)技能
4. **术语转换**: 对提示词主体执行标准转换流程
5. **输出结果**: 输出二级提示词（不含暗号）

无

## 技术约束

1. 暗号必须是单独字母，与前面内容有空格分隔
2. 暗号不区分大小写（`i` 和 `I` 均可，`v` 和 `V` 均可）
3. 如果用户输入仅为暗号（无实际内容），应提示用户输入内容
4. 暗号识别优先于其他触发条件
5. 输出的二级提示词不包含暗号部分
