# Checklist - 摄影术语扩展

## 验收检查点

- [x] **镜头运动术语**: 包含至少 12 组镜头运动术语映射（升格/降格/推/拉/摇/移/跟/甩/升/降/旋转/轻微运镜）
- [x] **景别术语**: 包含至少 10 组景别术语映射（特写/近景/中景/全景/远景/全身/半身等）
- [x] **拍摄角度术语**: 包含至少 10 组拍摄角度术语映射（俯视/平视/仰视/正面/侧面/背面等）
- [x] **焦段术语**: 包含至少 8 组焦段术语映射（35mm/50mm/75mm/85mm等）
- [x] **摄影风格术语**: 包含至少 8 组摄影风格术语映射
- [x] **完整转换示例**: 包含带摄影术语的完整转换示例
