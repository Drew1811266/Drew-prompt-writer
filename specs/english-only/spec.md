# 英文提示词纯度修复规格

## Why

在测试中发现，输出的英文提示词段落中混入了中文内容（如 `[参考图1中的...]`），这不符合输出规范。英文提示词部分应该完全使用英文，不得混入任何中文或中文括号注释。

## What Changes

- **修复英文提示词纯度**: 英文提示词段落不得包含任何中文内容
- **明确括号使用规则**: 图片引用在英文提示词中使用英文括号和英文描述
- **更新 SKILL.md**: 修正输出格式规范和示例

## Impact

- Affected specs: output-format
- Affected code: `.trae/skills/prompt-crafter/SKILL.md`

---

## ADDED Requirements

### Requirement: 英文提示词纯度规则

**WHEN** 输出英文提示词时
**THEN** 英文提示词段落必须完全使用英文，不得包含任何中文内容

**禁止出现**:
- 中文文字
- 中文括号 `【】`、`（）`
- 混杂中英文的描述（如 `[参考图1中的牛仔裤]`）

**正确格式**:
- 图片引用使用英文括号和英文描述
- 示例：`[floral dress from image 1]` 而非 `[参考图1中的连衣裙]`

### Requirement: 图片引用格式规范

**WHEN** 在英文提示词中包含图片引用时
**THEN** 必须遵循以下格式：

**正确格式**:
```
wearing [jeans from image 1]
holding [bubble tea from image 2]
```

**错误格式（禁止）**:
```
wearing [参考图1中的牛仔裤]
holding [图2中的奶茶]
```

### Requirement: 输出格式修正

**WHEN** 输出最终结果时
**THEN** 英文提示词段落必须完全英文，中文翻译段落才包含中文

---

## 技术约束

1. 英文提示词段落使用英文括号 `[]`
2. 图片引用描述使用英文（如 "from image 1" 而非 "中的图1"）
3. 所有主体、场景、技术参数描述均为英文
4. 中文翻译段落用于用户理解，可包含中文原文对应
