# 导演思维创作规格

## Why

用户需要 AI 在生成提示词时拥有"导演思维"——不仅是被动转换用户的一级提示词，而是主动进行创作性发挥。AI 应该像导演一样，理解用户的核心意图后，在用户规定的框架内主动补充画面细节、氛围营造、光影设计等，把一个简单的描述打造成一个完整的、有故事感的画面。

## What Changes

- **新增导演思维原则**: AI 创作提示词时应像导演一样主动思考画面
- **新增画面叙事性**: 提示词应包含故事感和情感表达
- **新增氛围营造**: 主动设计光影、色彩、情绪氛围
- **新增细节补充**: 基于用户描述创造性地添加环境细节
- **更新 SKILL.md**: 将导演思维创作规则整合到技能描述中

## Impact

- Affected specs: intelligent-expansion
- Affected code: `.trae/skills/prompt-crafter/SKILL.md`

## ADDED Requirements

### Requirement: 导演思维原则

**核心思想**:
AI 在创作提示词时应像电影导演一样思考：

1. **理解核心意图**: 理解用户真正想表达的是什么（不只是表面的词块）
2. **画面完整性**: 思考如何让画面更完整、更有故事感
3. **主动创作**: 不只是被动转换，而是主动补充细节
4. **保持一致性**: 所有创作必须符合用户设定的基本框架

### Requirement: 导演创作维度

**WHEN** AI 创作提示词时
**THEN** 应从以下维度进行主动创作：

| 创作维度 | 说明 | 示例 |
|----------|------|------|
| 光影设计 | 主动设计光线方向、色温、对比度 | 自然光、戏剧光、剪影等 |
| 色彩基调 | 主动设定整体色彩风格 | 暖色调、冷色调、对比色等 |
| 情绪氛围 | 主动营造画面情绪 | 温馨、神秘、活力等 |
| 环境细节 | 主动添加环境元素 | 风、草、水面反射等 |
| 镜头语言 | 主动设定视角和距离 | 远景交代、中景叙事、特写强调 |
| 故事感 | 主动添加动态或静态叙事元素 | 正在发生的故事、瞬间定格 |

### Requirement: 导演创作示例

**用户输入**:
```
女孩 公园 散步
```

**被动转换（无导演思维）**:
```
young girl walking in park
```

**导演思维创作后**:
```
young girl, 8 years old, red dress, blonde curly hair, walking leisurely through a sunlit park,
golden hour sunlight casting warm glow through autumn leaves,
catches a falling maple leaf, curious expression, gentle smile,
background of ancient oak trees, green grass, stone bench nearby,
cinematic atmosphere, nostalgic and peaceful mood,
wide establishing shot, depth of field, soft bokeh,
film-like quality, warm color grading
```

### Requirement: 创作约束

**WHEN** AI 进行导演思维创作时
**THEN** 必须遵守以下约束：

1. **不违背用户设定**: 所有创作必须与用户的一级提示词保持一致
2. **不添加冲突元素**: 如果用户设定"夜晚"就不能添加"阳光明媚"
3. **合理扩展**: 扩展内容应符合场景逻辑
4. **用户可控**: 如果用户描述已经足够详细，不过度创作

### Requirement: 导演创作时机

| 用户输入复杂度 | AI 创作程度 |
|----------------|-------------|
| 只有主体描述（如"女孩"） | 全面创作，补充场景、氛围、光影 |
| 主体+场景（如"女孩 公园"） | 重点创作氛围、光影、动态细节 |
| 主体+场景+动作（如"女孩 公园 散步"） | 适度创作，补充光影、环境细节 |
| 描述详细（如包含摄影术语） | 最小创作，保持用户设定为主 |

### Requirement: 创作输出模板

**导演思维创作的提示词结构**:
```
(主体) [用户描述的主体] + [导演补充的细节]
(场景) [用户描述的场景] + [导演补充的环境元素]
(光影) [导演设计的光线效果]
(色彩) [导演设定的色彩基调]
(氛围) [导演营造的情绪]
(动态) [导演添加的故事感元素]
(技术) [用户指定的摄影参数]
(质量) [质量标签]
```

## 技术约束

1. 导演创作是"智能扩展"的延伸，在智能扩展基础上进一步创作
2. 创作时应优先保留用户明确指定的所有元素
3. 创作内容应帮助生成更生动、更有故事感的画面
4. 总词数仍需控制在 800 词以内
